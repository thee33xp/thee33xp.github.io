DC Motor Project

Items Needed:

Raspberry Pi 4b+
Running the basic desktop port on the Pi with a head setup for code execution.
DC Motor
Simple DC Motor. Amazon or Hobby Lobby like $0.50.
Wires
Basic wires, probably 24ga? Also, 2 resistors.
External Power Supply
Mine came with a kit from Amazon, it puts out 5v or 3v.
Motor Driver
Specifically the L293D I.C. for this project.
Bread Board
Optional, but useful for drafting circuits.
GPIO Expansion Ribbon
Optional, but helpful for exapnding gpio pins to the board.
L293D reference
We're only using the red circled pins on the chip

Project Steps

Step One:
Place the L293D chip on the board
Step Two:
Chip Pin 1 to Pi Pin 17 & Chip Pin 2 to Pi Pin 27
Step Three:
Chip Pin 16 to Pi 5v Pin & Ground Pi/Chip Pins 4/5 to board
Step Four:
Add the External Power to the circuit
Step Five:
Connect Motor to Chip Pin 3 and ground to board
Final Result
Now to add the code!

Writing the Code

import pygame #using pygame for KEYS event handler

import RPi.GPIO as GPIO

pygame.init()

SCREEN = pygame.display.set_mode((420,333))

CLOCK = pygame.time.Clock()

ENABLE_PIN = 17

INPUT1_PIN = 27

GPIO.setmode(GPIO.BCM)

GPIO.setup(ENABLE_PIN, GPIO.OUT)

GPIO.setup(INPUT1_PIN, GPIO.OUT)

GPIO.output(ENABLE_PIN, GPIO.LOW)

def motor_forward():

GPIO.output(INPUT1_PIN, GPIO.HIGH)

GPIO.output(ENABLE_PIN, GPIO.HIGH)

def motor_stop():

GPIO.output(ENABLE_PIN, GPIO.LOW)

def HANDLER():

for event in pygame.event.get():

if event.type == pygame.QUIT or (event.type == pygame.KEYDOWN and event.key == pygame.K_ESCAPE):

GPIO.cleanup()

pygame.quit()

if event.type == pygame.KEYDOWN:

if event.key == pygame.K_UP:

motor_forward()

print("Motor Running")

elif event.key == pygame.K_DOWN:

motor_stop()

print("Motor Stopping")

def LOOP():

while True:

CLOCK.tick(30)

SCREEN.fill("DarkBlue")

HANDLER()

pygame.display.flip()

if __name__ == "__main__":

LOOP()
