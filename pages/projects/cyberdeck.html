CyberDeck Volume 1.0

Build

Raspberry Pi 4b+
Running the basic desktop port on the Pi with a head setup for code execution.
Case
Just using a basic waterproof case. I think I got mine at Harbor Freight.
Portable Battery Pack
This is a simple pack, I'd recommend getting one with more than just a single port..
7in Monitor
Not a must; but helpful if running multiple tasks or switching radio channels, etc.
Software Defined Radio
Cheap but effective tool for radio wave monitoring.
Bluetooth Keys
Not great for writing programs but great for simple navigation or code execution.
ELM327
Cheap tool for OBD monitoring.

Final Review

Bewarned! This is not battle-tested... Yet.

Honestly though, this has been a enjoyable first draft. I think it's really cool I can go anywhere my black-box of doom lol. Typically I bring along my larger Bluetooth Keyboard if I plan on doing any real coding but for simple radio monitoring or pre-programmed tasks I can get by with the smaller keys. I don't have any velcro holding everything in place yet but that would be useful for when the box is closed. The ELM device can come in really handy if using the pyOBD module and don't want to pay for a high-priced scanner.

Besides the impractical nature of building this, if you were going to build your own, I'd recommend using a battery bank with two ports as to not overdraw current from the pi. Right now, I've gotten away with a single port bank and running the LED off the pi but the wiring config is tricky; I went through a few wires before not getting a voltage warning.
